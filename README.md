- Pour lancer la bdd

```bash
docker-compose up -d
```
- Pour lancer l'app
```bash
python3 -m venv .venv
```
```bash
source .venv/bin/activate
```
```bash
pip install -r requirements.txt
```
```bash
python app.py
```
- puis se rendre sur localhost:5000
